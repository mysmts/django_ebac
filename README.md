# Projeto Django - EBAC

Este é um projeto inicial criado com Django como parte do curso da EBAC (Escola Britânica de Artes Criativas e Tecnologia).

## Objetivo

Configurar a estrutura base do projeto com Django, versionamento via Git e colaboração via GitHub, com revisão de código por professores da EBAC.

## Tecnologias Utilizadas

- Python 3.x
- Django 4.x
- Git & GitHub
